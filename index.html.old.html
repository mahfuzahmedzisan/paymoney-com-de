Installation

 Total Installation process done by following four steps.

1. Create a database in your server.

2. Upload payMoney source codes to your sever.

3. Input Database information .

4. Input Login details to set login information for this site.

2. Check folder permission-

You must give permission to some directories. (For more details visit - https://docs.paymoney.techvill.net/how-to-give-permission-on-you-sever/ )

3. Verify Envato Username & Purchase Code

4.If purchase code is verified, you will see the Database setting page.

Enter your database credentials.

Once you have completed filling credentials, please wait for few seconds.

You will be passed to next step; otherwise, please re-check your credentials and hostname. (usually hostname is localhost but you should check with your hosting provider or cPanel.

Input a administrator name , email address and password and submit to complete installation.

Name: Set your administrator name here.

Email: Set email address as username.

Password: Input your password

If the installation is successful, you will see the message Installation Successful.
